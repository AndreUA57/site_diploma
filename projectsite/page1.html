Project I make

This diploma project presents the development of an information system for an online toy store. Its main goal is to simplify and automate the processes of managing products, customers, and orders. The system provides convenient tools both for users who want to shop for toys and for administrators who manage the store‚Äôs content and operations.

For customers, the system offers a modern, user-friendly interface where they can browse toys, search by categories or keywords, view product details, and place orders. The system is built using modern web technologies and a relational database to ensure stability, data security, and scalability. Backend and frontend parts are well-structured to provide a seamless experience for all users.
